# Dashboard de Criptomoedas

Este é um projeto de dashboard desenvolvido em Python com Dash, onde você pode acompanhar informações sobre diversas criptomoedas, como preço atual, capitalização de mercado, volume de negociação e variação percentual nas últimas 24 horas.

## Funcionalidades

- Visualização do preço atual, capitalização de mercado, volume de negociação e variação percentual das criptomoedas selecionadas.
- Gráficos de histórico de preços e volume de negociação para análise de tendências.
- Seleção de criptomoeda e moeda de cotação para personalizar a visualização dos dados.

## Tecnologias Utilizadas

- Python
- Dash
- Dash Bootstrap Components
- Plotly

## Como Usar

1. Clone este repositório:

git clone https://github.com/natatgh/cript-coin-dash

2. Instale as dependências necessárias:

pip install dash dash_bootstrap_components requests pandas plotly

3. Execute o aplicativo:
   
python app.py

4. Abra o navegador e acesse o endereço indicado no terminal.
